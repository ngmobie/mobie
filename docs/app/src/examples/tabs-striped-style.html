Add tabs-striped to an element above the tabs classname for Android style tabs. Optionally, also add tabs-top to position the tab at the top Get granular color control for striped tabs with the tabs-background-{color} and tabs-color-{color} classes, where {color} is any of the ionic color swatches: default, danger, info, success, primary Note, that to have the header blend with the top tabs, add the has-tabs-top class to the header.
